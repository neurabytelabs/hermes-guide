# Hermes Zero-to-Hero

> From Chat User to Agent Operator

A visual guide to using [Hermes](https://github.com/hermes-ai/hermes) correctly — with durable context, inspection before action, planning before change, and workflows that compound.

**Live:** [hermes-guide.neurabytelabs.com](https://hermes-guide.neurabytelabs.com)

## What This Is

Most people use AI like a chatbot. This guide teaches a different model:

- **Inspect** the current state before acting
- **Plan** changes before making them
- **Act** deliberately with real tools
- **Compound** what works into reusable systems

The guide includes an embedded **ASCII Cinema** — a living, animated walkthrough of the full operational loop rendered in ASCII art.

## Features

- Interactive ASCII cinema with 7 narrative scenes
- Scene-aware background palettes
- Living agent animation
- Play/pause/restart controls
- Light and dark theme support
- Responsive layout
